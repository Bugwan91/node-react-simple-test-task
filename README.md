Server
====
Dependencies
----
- Node.js

run command
- `node <path_to_server_directory>/index.js`

Client
====
Dependencies
----
Node.js
React

Install
----
- go to client folder: `cd <path_to_client_directory>`
- install all dependencies `npm install`
- run client: `npm start`

Info
----
Client work on port 3000, Server on 3001
